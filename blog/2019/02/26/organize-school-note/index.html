やったことのメモ

サブディレクトリに別のリポジトリを入れる

目的

別々のリポジトリで管理していたものを一つに統合する

手順

  1. サブディレクトリに入れるリポジトリrepo-src について
    1. git remote add repo-src /path/to/repo-src
    2. git fetch repo-src
  2. git subtree add --prefix=path/to/dist repo-src master
